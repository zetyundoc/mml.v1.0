##应用服务器推荐配置
应用服务器操作系统支持Linux内核版本2.6以上，以及Windows Server 2008 R2以上版本，应用服务器需安装1.7以上版本JAVA环境。

##采集端系统兼容性
MML采集端需运行Java 1.7+以上版本。

##Web浏览器兼容性
MML支持Chrome,Firfox,Edge最新版和IE10以上版本的多种浏览器。
